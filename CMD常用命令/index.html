用惯Linux就忘掉CMD了

CMD常用命令

命令文档

00 系统命令

  • whoami 查看当前用户
  • hostname 显示主机名
  • ipconfig 查看网卡信息
  • help 查看帮助
  • systeminfo 查看系统信息
  • ver 显示Windows版本
  • date 显示或设置日期
  • time 显示或设置时间
  • cls 清屏
  • shutdown /s 关机
  • shutdown /s /t n 定时关机（时间n秒）
  • shutdown /a 取消关机
  • shutdown /r 重启
  • shutdown /l 注销
  • shutdown /h /f 休眠
  • win+E 打开文件管理器
  • win+D 显示桌面
  • win+L 锁定

01 目录与文件操作

  • d: 进入d盘
  • cd 切换目录 / 显示当前目录
  • cd .. 返回上级目录
  • cd / 返回根目录
  • dir 查看当前目录的内容
  • md / mkdir 创建目录
  • rd / rmdir 删除目录
  • type 显示文本文件内容
  • ren 重命名文件
  • del 删除文件
  • find 查找文件
  • copy 复制文件
  • move 移动文件

02 进程操作

  • tasklist 查看进程
  • tasklist /svc 查看进程对应的服务
  • taskkill 杀死进程
  • findstr 查找字符串，类似grep，可以结合管道|使用

03 net命令

  • net user 查看用户信息
  • net user 用户名 密码 /add 添加用户
  • net user 用户名 /delete 删除用户
  • net localgroup 组名 用户名 /add 将用户添加到组中
  • net 用户名 密码 更改用户密码
  • net 密码 更改登录密码
  • net localgroup 查看本地组
  • net start 查看开启的服务
  • net start 服务名 启动服务
  • net pause 服务名 暂停服务
  • net continue 服务名 继续服务
  • net stop 服务名 关闭服务
  • net config 查看系统的网络设置
  • net share 查看开启的共享

04 其他网络命令

  • arp 查看arp缓存
  • netstat 查看端口信息
  • ping
  • tracert 查看与目标直接的路由路径
  • telnet 远程登录
