Linux和vim最常用的一些命令

Linux基本命令

01 系统命令

  • whoami 查看当前用户
  • ifconfig 查看网卡信息
  • netstat 查看端口信息
    • -a 列出所有端口
    • -t TCP端口
    • -u UDP 端口
    • -l 查看监听端口
    • -s 统计端口信息
    • -p 显示PID和进程名称
    • -n 不显示主机、端口和用户名，用英文代替
    • -c 每秒持续输出
    • -r 显示核心路由信息
    • -i 显示网络接口列表
  • history 查看命令历史记录
  • clear 清屏
  • alias 为命令设置别名
    • alias lx=ls 将lx设置为ls的别名
  • –help 查看帮助信息
  • man 查看手册
  • > 将输出结果保存到指定文件（覆盖）
  • >> 将输出结果保存到指定文件（在尾部追加）
  • | 管道，将前面命令的输出结果作为后面命令的输入
  • reboot 重启
  • shutdown 关机
    • -h 指定时间
      • -h now 立即关机
      • -h +10 十分钟后关机
  • 常用快捷键
    • tab 自动补齐
    • Ctrl+U 清空至行首
    • Ctrl+K 清空至行尾
    • Ctrl+L 清屏
    • Ctrl+C 取消命令编辑或执行

02 目录与文件操作

  • pwd 查看当前目录

  • cd 切换目录

    • cd+绝对路径
    • cd+相对路径
    • cd ~ 返回用户目录
    • cd . 不切换，仍然在当前目录
    • cd .. 切换到上级目录
    • cd ../.. 切换到上上级目录
    • cd - 切换到上一次访问的目录

  • ls 查看文件与文件夹

    • ls -a 查看所有文件与文件夹（包含隐藏内容）
    • ls -l 以列表的形式显示详细信息（包括权限信息、时间、大小等）
    • ls -h 配合-l使用，将大小以k为单位显示
    • ls -R 递归列出子文件夹与子文件

  • tree 查看树状结构

  • find 查找文件

  • touch 创建文件

    • touch+文件名 创建单个文件，可以用空格分开，同时创建多个文件
    • touch+文件名{起始字符..终止字符} 批量创建文件
      如 touch test{a..d} 就创建了testa,testb,testc,testd四个文件

  • mkdir 创建文件夹

    • mkdir -p 创建多个文件

  • mv 移动并重命名文件，文件夹

    • mv 要重命名的文件 新文件名
    • mv 要移动的文件名 目标目录
    • mv -i 询问移动时是否覆盖同名文件
    • mv -b 不覆盖，将原来的同名文件备份
    • mv -f 强制覆盖

  • cp 复制文件

    • cp -i 询问复制时是否覆盖同名文件
    • cp -f 强制覆盖
    • cp -r 复制文件夹及其子文件夹和文件

  • cat 查看文件内容

  • more 分页查看文件内容

  • less 分行查看文件内容

  • head -n 查看文件前n行，默认为10

  • tail -n 查看文件后n行，默认为10

  • rm 删除文件，文件夹

    • rm -i 删除时进行询问
    • rm -f 强制删除
    • rm -r 删除文件夹

  • ln 建立链接

    • ln 源文件 链接文件 硬链接，占用硬盘空间
    • ln -s 源文件 链接文件 软链接，不占用硬盘空间

  • grep 用于过滤搜索指定的字符串

    • -a \ –text 不忽略二进制的数据
    • -c \ –count 计算查找结果的列数
    • -E <匹配样式>\ –extended-regexp=<匹配样式> 使用正则表达式匹配
    • -i \ –ignore-case 忽略大小写
    • -n \ –line-number 显示匹配结果的列的编号
    • -v \ –revert-match 显示不匹配的所有行

  • tar 压缩和解压

    • -c / –create 创建新的备份文件

    • -C 在特定的目录解压缩

    • -v 显示操作过程

    • -f <文件名> / –file=<文件名> 指定备份文件

    • -x / –extract / –get 从备份文件中还原文件

    • -t / –list 列出备份文件的内容

    • -z 打包后，以 gzip 压缩

    • -j 打包后，以 bzip2 压缩

    • 常用命令

      • 压　缩：tar -jcvf filename.tar.bz2 要被压缩的文件或目录名称
      • 查　询：tar -jtvf filename.tar.bz2
      • 解压缩：tar -jxvf filename.tar.bz2 -C 欲解压缩的目录

03 账号与权限

  • 用户账号分类（标识符UID）

    • 超级用户
    • 程序用户
    • 普通用户

  • 组账号分类（标识符GID）

    • 基本组
    • 附加组

  • 信息存储

    • /etc/passwd 保存了用户信息
    • /etc/shadow 保存了密码信息
    • /etc/group 保存了组信息

  • useradd 添加用户

    • -d 指定用户主目录，如果不存在就使用-m选项新建目录
    • -g 指定所属的用户组
    • -G 指定所属的附加组
    • -s 指定登录shell
    • -u 指定用户号
    • -o 可以重复使用其他用户的用户号

  • userdel 删除用户

    • -r 将用户的主目录一起删除

  • usermod 修改用户信息，选项与useradd相似

  • passwd 修改密码（超级用户可以修改其他用户的密码）

    • -l 禁用账号
    • -u 口令解锁
    • -d 使账号口令为空
    • -f 使用户下次登录时修改口令

  • groupadd 添加组

    • -g 指定组标识号
    • -o 可以重复使用其他组的组号
    • -n 将组的名字改为新名字

  • newgrp 组名 切换到其他组

  • 文件类型

    • d 目录
    • - 文件
    • l 链接文件
    • b 可控存储的接口设备
    • c 串行端口设备

  • 文件权限

    • r 读取，对应数字4
    • w 写入，对应数字2
    • x 执行，对应数字1

  • chgrp 属组名 文件名 更改文件属组

    • -R 递归更改

  • chown 属主名(:属组名) 文件名 更改属主（和属组）

    • -R 递归更改

  • chmod 三位数字 文件或目录 更改权限为数字对应的权限

    • -R 递归更改

  • chmod u=rwx,g=rx,o=r 文件名 属主设为rwx，属组为rx，其他为r

    • u表示属主，g表示属组，o表示其他
    • + 增加权限、- 去除权限、=设定为权限

04 进程

  • ps 查看进程
    • a 查看所有程序，包括其他用户的程序
    • u 用以用户为主的格式显示
    • x 显示所有程序，不以终端机来区分
    • c 显示每个程序真正的指令名称
    • e 显示每个程序的环境变量
    • -A 显示所有程序
    • -H 显示树状结构（pstree）
    • -N 显示所有程序，除了执行ps指令的终端机下的程序
    • -t 指定终端机编号
  • pgrep 查找进程
    • -o 显示最小的进程号
    • -n 显示最大的进程号
    • -l 显示进程名称
    • -p 指定父进程好
    • -g 指定进程组
    • -t 指定终端
    • -u 指定用户ID
  • kill 终止进程号对应的进程
    • -9 强制终止
  • top 查看当前进程的资源占用情况
    • -u 指定用户名
    • -p 指定进程号
    • -c 显示完整命令
    • -n 设置更新次数
    • -d 设置更新间隔，单位秒
  • 进程操作
    • Ctrl+z 将前台进程调入后台
    • Ctrl+c 强制结束进程
    • jobs 查看后台进程
    • fg n 将后台第n个进程在前台运行
    • bg n 将后台第n个进程在后台运行
    • & 将进程放在后台运行

05 计划任务

  • crontab 周期性计划任务

    • 配置文件 /etc/crontab

    • 启动服务 service crond start

      • 类似的还有stop,restart,reload,status

    • 查看是否为开机启动 ntsysv

      • 设置为开机启动ntsysv -level 35 crond on

    • 配置内容（文件前四行）

      • SHELL 使用的shell
      • PATH 执行命令的路径
      • MAILTO 将执行信息使用电子邮件发送给root用户
      • HOME 使用的主目录

    • 执行命令的格式

      • minute hour day month week command

      • 各字段可以用以下特殊符号

        • * 星号代表所有可能的值
        • , 逗号分隔多个取值
        • - 表示一个整数范围（包括两端）
        • / 表示间隔，如hour字段的0-23/2 表示每天每两小时执行一次

    • crontab -e [-u 用户名] 编辑计划任务

    • crontab -l [-u 用户名] 查看计划任务

    • crontab -r [-u 用户名] 删除计划任务

      • 如果编辑时没有指定用户名表示为当前用户设定计划任务

  • at 执行一次性任务

    • 命令格式 now + count time-units 表示从现在开始过多久执行任务

    • -f 指定包含具体指令的任务文件

    • -q 指定新任务的队列名称

    • -l 显示待执行任务的列表

    • -d 删除指定的待执行任务

    • -m 任务执行完成后向用户发送

    • atq 查看当前的任务列表

    • atrm n 删除编号为n的任务

    • at -c n 查看编号为n的任务的内容

06 vim

  • 三种模式的转换

    • 命令模式==》末行模式 ：键
    • 命令模式==》输入模式 a,i,o键
    • 输入模式、末行模式==》命令模式 Esc键

  • 命令模式操作

    • 光标移动
      方向键或k、j、h、l
    • 翻页
      • 向下翻页 PageDown 或 Ctrl+F
      • 向上翻页 PageUp 或 Ctrl+B
    • 行内跳转
      HOME键、^键、数字0
    • 行间跳转
      • 1G、gg跳转到首行
      • G 跳转到末行
      • nG 跳转到第n行
    • 剪切（删除）
      • x、Del键 删除当前光标处的单个字符
      • dd 剪切当前行
      • ndd 剪切从光标处开始的n行
      • d^ 删除当前光标到行首的所有内容
      • d$ 删除当前光标到行尾的所有内容
    • 复制
      • yy 复制当前行
      • nyy 复制从光标处开始的n行
    • 粘贴
      • p 粘贴到当前行下
      • P 粘贴到当前行上
    • 撤销
      • u 取消最近一次操作
      • U 取消当前行的所有操作
    • 保存
      ZZ 保存并退出当前编辑

  • 末行模式

    • 查找

      • /word 从上至下在文中查找word字符串
      • ?word 从下至上在文中查找word字符串
      • n 定位下一个匹配字
      • N 定位上一个匹配字

    • 显示行号

      • :set nu 显示行号
      • :set nonu 取消显示行号

    • 保存

      • :w 保存修改内容
      • :w /path/file 另存为

    • 退出

      • :q 未修改退出
      • :q! 放弃修改强制退出

    • 保存并退出

      :wq! 保存退出

    • 打开新文件编辑

      :e /path/file

    • 读入文件内容到当前编辑

      :r /path/file

    • 将当前行查找到的第一个字符串old替换为new

      :s /old/new

    • 将当前行查找到的所有字符串old替换为new

      :s /old/new/g

    • 在行号范围 #,#内查找到的所有字符串old替换为new

      :#,# s /old/new/g

    • 在整个文件内查找到的所有字符串old替换为new
      :% s old/new/g

    • 在替换命令末尾加上c，用户将对每次替换进行确认
