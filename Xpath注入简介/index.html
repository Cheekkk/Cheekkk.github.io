xpath语法和注入漏洞简介

Xpath

01 Xpath

  1. 简介
    Xpath使用路径表达式来选取 XML 文档中的节点或者节点集

    <?xml version="1.0" encoding="UTF-8"?>

<bookstore>
  <book>
    <title lang="en">Harry Potter</title>
    <author>J K. Rowling</author>
    <year>2005</year>
    <price>29.99</price>
  </book>
</bookstore>

    在 XPath 中，有七种类型的节点：元素、属性、文本、命名空间、处理指令、注释以及文档（根）节点。XML 文档是被作为节点树来对待的。树的根被称为文档节点或者根节点

    <bookstore>是文档节点
    <author>J K. Rowling</author>是元素节点
    lang=”en”是属性节点

  2. 语法

    可以参考菜鸟教程

02 Xpath注入

  1. 原理

    Xpath注入与SQL注入较为类似，只是改为了从xml文档中获取数据，注入的方法仍然是拼接查询语句

  2. 实例
    创建一个简单的xml文档验证一下

    <!--book.xml-->

<bookstore>
    <book>
        <name>book-1</name>
        <author>Alice</author>
        <price>10</price>
    </book>
    <book>
        <name>book-2</name>
        <author>Bob</author>
        <price>20</price>
    </book>
    <book>
        <name>book-3</name>
        <author>Eve</author>
        <price>30</price>
    </book>
</bookstore>

    查询页面

    //bookguide.php

<?php
$xml = simplexml_load_file('book.xml');
$id = $_GET['id'];
$query = "/bookstore/book[id/text()='".$id."']";
echo $query;
$result = $xml->xpath($query);
if($result) {
    echo '<h2>Result:</h2>';
    foreach ($result as $key => $value) {
        echo 'Author:'.$value->author;
        echo '<br />';
        echo 'Price:'.$value->price;
    }
}

    传入参数id即可查询对应的信息

    查询出了对应的信息
